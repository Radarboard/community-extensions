# Radarboard Community Extensions

Community-maintained integrations, plugins, and widgets for Radarboard.

This repository is intentionally separate from the core Radarboard monorepo so
community extensions can evolve with their own review, validation, and release
flow.

## Structure

- `integrations/` for third-party service connectors.
- `plugins/` for full product surfaces.
- `widgets/` for dashboard cards and panels.
- `scripts/` for catalog generation and validation.

## Status

This repository starts fresh under the `Radarboard` organization. Legacy commits,
tags, and releases were not migrated.
